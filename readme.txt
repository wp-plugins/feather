== Description ==

Feather is a lightweight and powerful WordPress theme framework. The framework allows you to easily configure your theme using the WordPress Dashboard and multiple configuration files.

== Installation ==

Manual Installation

1. Upload feather to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a _feather_ directory in your theme's directory.
